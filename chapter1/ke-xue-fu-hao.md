#### 指數律六大公式：

a的m次方 x a的n次方 = a的m+n次方

a的m次方 / a的n次方 = a的m-n次方

\(a的m次方\)的n次方 = a的mxn次方

a的n次方xb的n次方 = \(axb\)的n次方

a的0次方 = 1，a不等於0

a的-n次方 = 1/a的n次方，a不等於0
